# PRODIGY_WD_02 - Stopwatch Web Application

##  Project Description
This project is a stopwatch web application that allows users to track time, pause, reset, and record lap times.

## Features
- Start / Pause / Reset functionality
- Lap time recording
- Milliseconds precision
- Responsive design
- Modern UI

## Technologies Used
- HTML
- CSS
- JavaScript
